# RunUI
World's first running app written in SwiftUI.  
Using Core Location and Core Data.  
Updated for xCode 11 GM  

* Distance measuring
* List (vertical, horizontal)
* NavigationView
* Forms
* Half Modal
* PickerView
* Sheet
* Alert, ActionSheet
* ObservableObject
* EnvironmentObject
* UIViewRepresentable
* Data fetching
* Custom Drawing
* Gestures
* DynamicProperty (@State, @ObservedObject, @EnvironmentObject, @GestureState, @Binding)
